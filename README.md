# Telegram Bot для профориентации школьников

Телеграм-бот для помощи школьникам в выборе профессии с использованием AI (Google Gemini).

## Возможности

- **Профориентационный тест** - 10 вопросов для определения склонностей
- **Информация о профессиях** - подробное описание 5 профессий (Программист, Инженер, Врач, Юрист, Экономист)
- **Анализ увлечений** - определение подходящей профессии на основе хобби
- **AI-консультант** - 5 бесплатных вопросов к искусственному интеллекту
- **База университетов** - информация о ведущих вузах Москвы

## Технологии

- Python 3.13
- aiogram 3.x (Telegram Bot API)
- SQLAlchemy 2.x (async)
- PostgreSQL 15
- Alembic (миграции БД)
- Google Gemini AI
- Docker & Docker Compose

## Быстрый старт

1. Клонируйте репозиторий
2. Настройте `.env` файл (см. SETUP.md)
3. Запустите контейнеры:
```bash
docker-compose up -d
```

Бот автоматически:
- Применит миграции БД
- Создаст таблицы
- Заполнит базу профессиями и университетами
- Запустится и будет готов к работе

## Команды бота

- `/start` - Начать работу с ботом

## Основные функции

### Профориентационный тест
10 вопросов с ответами "Да/Нет" для определения склонности к профессиям:
- 0-3 балла: Юрист, Медик
- 4-7 баллов: Экономист, Инженер
- 8-10 баллов: Программист, Инженер

### Профессии
Информация о 5 профессиях с данными о:
- Описании профессии
- Необходимых предметах для ЕГЭ
- Подходящих университетах

### Анализ увлечений
Определение профессии на основе ключевых слов из описания хобби пользователя.

### AI-консультант
5 бесплатных вопросов к Google Gemini AI (лимит на пользователя).

## База данных

### Таблицы:
- `users` - пользователи (tg_id, points, ai_questions_count)
- `professions` - профессии
- `universities` - университеты
- `professions_universities` - связь профессий и университетов
